# VIBE-X 프로젝트 정의서

> **팀 바이브제왕** | 버전: v1.0 | 최종 수정: 2026.02.12

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
**VIBE-X** — Agent + MCP + RAG + Collaboration 통합 협업 플랫폼

### 1.2 비전
바이브 코딩의 구조적 한계(모델 품질 격차 + 팀 협업의 벽)를 시스템으로 해결하여,
개인의 AI 코딩 도구를 **팀의 협업 인프라**로 진화시킨다.

### 1.3 핵심 가치
- **구조화** — 암묵적 지식을 명시적 시스템으로 전환
- **자동화** — 반복적 검증과 기록을 Agent에게 위임
- **영속성** — 휘발성 지능을 영속적 지식으로 변환
- **협업** — 개인 최적화를 팀 최적화로 확장

## 2. 기술 스택

### 2.1 Core
| 영역 | 기술 | 버전 | 선정 이유 |
|------|------|------|----------|
| 언어 | TypeScript | 5.x | 타입 안전성, 프론트/백엔드 통합 |
| 런타임 | Node.js | 20 LTS | 장기 지원, 생태계 |
| 패키지 관리 | pnpm | 9.x | 빠른 속도, 디스크 효율 |

### 2.2 Backend
| 영역 | 기술 | 선정 이유 |
|------|------|----------|
| API 서버 | FastAPI (Python) | RAG/ML 파이프라인 연동 용이 |
| 보조 서버 | Express.js | MCP 서버, WebSocket 처리 |
| Vector DB | LanceDB | 임베디드 모드, 무료, 빠른 속도 |
| 임베딩 | Voyage-code-3 | 코드 이해도 최상위 |
| Orchestration | LangChain | RAG 파이프라인 표준 |

### 2.3 Frontend
| 영역 | 기술 | 선정 이유 |
|------|------|----------|
| 프레임워크 | Next.js 14+ | App Router, SSR, 모던 대시보드 |
| 스타일링 | TailwindCSS 4 | 유틸리티 퍼스트, 빠른 개발 |
| 차트 | Chart.js | 경량, 반응형, 다양한 차트 |
| 상태 관리 | Zustand | 심플, 보일러플레이트 최소 |

### 2.4 Infra / DevOps
| 영역 | 기술 | 선정 이유 |
|------|------|----------|
| Git Hooks | Husky | 커밋 시점 자동 검증 |
| 린터 | ESLint + Prettier | 코드 품질 표준화 |
| CI/CD | GitHub Actions | 무료, Git 네이티브 통합 |
| 통신 | MCP (Model Context Protocol) | Agent 간 표준 통신 |

### 2.5 IDE
| 영역 | 기술 | 선정 이유 |
|------|------|----------|
| 주 IDE | Cursor | MCP 연동, AI 네이티브 |
| 보조 IDE | VSCode + Continue.dev | 오픈소스 RAG 지원 |

## 4. 마일스톤 및 릴리즈 계획

| 마일스톤 | 기간 | 핵심 산출물 | 비용 |
|---------|------|-----------|------|
| **M1: Foundation** | Day 1 (3h) | 팀 문서 체계, PACT-D, .cursorrules | $0 |
| **M2: RAG Engine** | Week 1~2 | Vector DB, 임베딩, 기본 품질 게이트 | $0 |
| **M3: Agent Chain** | Month 1~2 | 6-Gate 자동화, MCP 오케스트레이터 | 저비용 |
| **M4: Platform** | Month 3~6 | 대시보드, IDE 플러그인, 온보딩 자동화 | 중비용 |

### 릴리즈 전략
- **v0.1** (M1 완료) — 문서 기반 운영, 수동 PACT-D
- **v0.5** (M2 완료) — RAG 검색 가능, 기본 Gate 자동화
- **v1.0** (M3 완료) — Agent 체인 완전 자동화
- **v2.0** (M4 완료) — 풀 플랫폼 + IDE 플러그인
